# 99 Numbers

A mobile-friendly pyramid number puzzle. Place cards from a shuffled stack of 1–99 so each row’s total fits between the rows above and below — before time runs out.

## Play

Open [the live game](https://bhagh.github.io/99-numbers/) or open `index.html` locally in a browser.

## Rules (short)

- Fill all 10 pyramid spaces within 60 seconds
- Valid placement: +3s · Invalid: −5s · Pass skips a card
- Win score: pyramid total + 5× seconds left + 1× cards left
- Time out / not enough cards: half the pyramid total
